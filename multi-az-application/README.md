# CVM 多可用区申请原型

## 当前原型

- `index.html`：CVM 实例申请页、多可用区分布、申请确认页、申请单详情页一体化交互原型

## 设计依据

- 保持 Panther 后台控制台现有视觉风格
- 页面尽量保持现状不变，以小改动支持多可用区创建
- 只支持同一机房下多个可用区，其他配置保持一致
- 网络继续统一为系统分配
- 1 张申请单内包含多个可用区分布

## 本页表达范围

- 默认金山云、默认单可用区创建、默认 B 区
- 单可用区和多可用区创建方式切换
- 单可用区和多可用区下的数量、分布联动
- 镜像、项目、实例配置、网络与带宽、其他设置的静态交互
- 申请确认页和申请单详情页的多可用区信息展示

## 暂不表达

- 后端真实提交和审核流转
- 可用区库存、售罄、权限等异常分支
- 实际实例创建结果回写
