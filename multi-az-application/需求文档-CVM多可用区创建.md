## 1 概述

| 项目 | 说明 |
| --- | --- |
| 文档简介 | 本文档用于说明 CVM 实例申请页支持多可用区创建的需求方案。 |
| 适用人员 | 产品、前端开发、后端开发、测试 |
| 需求背景 | 当前 CVM 实例申请页只能通过底部数量批量创建同配置实例，无法表达“同一套配置分布到多个可用区分别创建”的场景。SCLB 高可用场景下，需要分别在 A、B 可用区创建实例，现有机制下只能拆成多次申请。 |
| 需求目标 | 在尽量保持页面现状不变的前提下，支持用户在 1 张申请单内完成多个可用区实例分布配置。 |
| 适用范围 | 本期支持金山云、亦庄的多可用区创建；媒体机房保持现状，不纳入本期多可用区改造范围。 |
| 原型页面 | `multi-az-application/index.html` |

## 2 需求详情

### 2.1 范围说明

| 项目 | 说明 |
| --- | --- |
| 本期支持 | 同一机房下多个可用区创建；多可用区之间仅允许“可用区”和“实例数量”不同；网络继续为系统分配；1 张申请单内包含多个可用区分布；只针对系统盘做可用区划分，数据盘不做划分。 |
| 本期不支持 | 跨机房创建；不同可用区使用不同镜像、规格、磁盘、网络等差异化配置；创建结果按可用区展示独立状态；媒体机房多可用区改造。 |

### 2.2 实例申请页

| 需求描述 | 原型说明 |
| --- | --- |
| 在“机房”下新增“创建方式”，支持“单可用区”和“多可用区”。 | 原型创建页基础配置区域 |
| 默认值：默认机房为金山云，默认创建方式为单可用区，默认可用区为 B 区。 | 原型默认进入态 |
| 单可用区时，保持原有可用区按钮形式；底部数量保持可编辑。 | 原型创建页单可用区模式 |
| 单可用区时，保留提示文案：`只针对系统盘做可用区划分，数据盘不做划分。` | 可用区下方提示 |
| 单可用区时，当数量大于 1 且当前机房支持多可用区时，在底部数量右侧显示引导文案：`如需高可用部署，建议切换为多可用区`。 | 底部汇总栏数量右侧提示 |
| 多可用区时，原可用区按钮区域替换为可用区分布表格。 | 原型创建页多可用区模式 |
| 当前原型中，多可用区固定为 A/B 两个可用区，不支持新增或删除分布项。 | 原型分布表格 |
| 多可用区时，每个可用区分别填写实例数量。 | A 区、B 区数量输入 |
| 多可用区时，底部数量保留但改为只读，总数自动汇总。 | 底部汇总栏 |
| 多可用区时，底部数量右侧显示：`请在上方可用区中调整实例数量`。 | 底部汇总栏数量右侧提示 |
| 媒体机房仍可选中，但选中后不展示“创建方式”，保持现有单可用区申请方式，不展示高可用引导文案。 | 媒体机房交互分支 |

### 2.3 申请确认页

| 需求描述 | 原型说明 |
| --- | --- |
| 在公共配置区展示创建方式、总实例数。 | 申请确认页公共配置区 |
| 新增“可用区分布”区块，展示机房、可用区、实例数量。 | 申请确认页可用区分布区块 |
| 不展示状态、备注、额外说明文案。 | 申请确认页展示范围 |

### 2.4 申请单详情页

| 需求描述 | 原型说明 |
| --- | --- |
| 在公共配置区展示创建方式、总实例数。 | 申请详情页公共配置区 |
| 新增“可用区分布明细”区块，展示机房、可用区、申请数量。 | 申请详情页可用区分布明细区块 |
| 不展示创建状态、备注。 | 申请详情页展示范围 |

### 2.5 关键交互

| 场景 | 交互说明 |
| --- | --- |
| 单可用区 -> 多可用区 | 隐藏原可用区按钮区域，展示 A/B 两区分布，底部数量自动汇总且不可编辑。 |
| 多可用区 -> 单可用区 | 隐藏分布表格，恢复单可用区按钮形态，底部数量恢复可编辑。 |
| 单可用区数量调整 | 底部数量表示当前所选可用区内创建的实例数量；当数量大于 1 时显示高可用引导。 |
| 多可用区数量调整 | A/B 两区分别填写数量；任一分布项数量变化后，底部总数量同步更新。 |
| 机房切换 | 金山云、亦庄支持创建方式切换；媒体机房隐藏创建方式，回到单可用区形态，不展示高可用引导。 |

## 3 验收要点

| 编号 | 验收项 | 验收标准 |
| --- | --- | --- |
| 1 | 默认态 | 默认进入申请页时，机房为金山云，创建方式为单可用区，可用区为 B 区。 |
| 2 | 单可用区 | 保留原有可用区选择方式和可编辑数量输入。 |
| 3 | 单可用区引导 | 数量大于 1 时，在底部数量右侧显示高可用引导文案。 |
| 4 | 多可用区形态 | 显示固定 A/B 两区分布。 |
| 5 | 多可用区数量 | 底部数量自动汇总且不可编辑。 |
| 6 | 多可用区提示 | 底部数量右侧显示：`请在上方可用区中调整实例数量`。 |
| 7 | 媒体机房 | 媒体机房可选，但不展示创建方式，不展示高可用引导。 |
| 8 | 申请确认页 | 新增可用区分布区块，不展示状态、备注、额外说明。 |
| 9 | 申请详情页 | 新增可用区分布明细区块，不展示创建状态、备注。 |
