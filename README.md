# CVM Prototype

CVM console prototype and interaction design.

## Prototype Modules

### `multi-az-application/`

CVM multi-AZ application flow with create page, confirmation page, and detail page.

- Unified prototype file: `multi-az-application/index.html`
- Scope:
  support single-AZ and multi-AZ creation under the same computer room, keep other configuration consistent, and show AZ distribution in one application form
